# **Aviation Safety Analysis**

## Project Overview:
This analysis identifies the safest aircraft manufacturers for Teroh Airways' new fleet acquisition strategy. Using historical accident data, we provide data-driven recommendations for aircraft purchasing decisions.

## Key Insights:

### 1. Safest Aircraft Manufacturers:
Identified the top 5 safest aircraft makes based on fatality percentage:
- **GRUMMAN ACFT ENG COR-SCHMEIZER** (1.7% fatality rate)
- **BOMBARDIER INC** (2.9% fatality rate)
- **STIMSON** (3.3% fatality rate)
- **BOEING** (3.5% fatality rate)
- **Raven** (3.5% fatality rate)

### 2. Safety Trends Over Time:
Analyzed aviation safety improvements from 2010-2022:
- Industry fatality rate decreased from 23.0% (2015) to 18.7% (2022)
- Top manufacturers maintain consistent safety records over time

### 3. Weather Resilience:
Evaluated manufacturer performance in different weather conditions:
- Top manufacturers are 74-91% safer than industry average in poor weather
